W4 Decision trees

  1. Introduction

    1. Decision 1: How to choose what feature to split node?

      Maximize purity （or minimize impurity）

    2. Decision 2: When do you stop splitting?

      • When a node is 100% one class
      • When splitting a node will result in the tree exceeding a maximum depth
      • When improvements in purity score are below a threshold
      • When number of examples in a node is below a threshold

  2. Entropy: information gain, we shoule choose lowest entropy

    p1p_1p1​: cats of animals to the side

    w1w_1w1​: animals to the size of all animals

  3. Process(Recursion Algorithm)

    1. Start with all examples at the root node
    2. Calculate information gain and pick the one with for all possible the highest information gain
    3. Split dataset according to selected feature, and create left and right branches of the tree
    4. Keep repeating splitting process until criteria is met(Decision 2)
